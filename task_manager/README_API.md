# Task Manager API

Basic REST API in Rust that manages tasks in memory. Each task has:

- `id` (generated by the server)
- `title` (string)
- `completed` (boolean)

## Run

```bash
cargo run
```

Server listens on `http://127.0.0.1:3000`.

## Endpoints

### Create

```bash
curl -X POST http://127.0.0.1:3000/tasks \
  -H "Content-Type: application/json" \
  -d '{"title":"Learn Rust"}'
```

### Read all

```bash
curl http://127.0.0.1:3000/tasks
```

### Read one

```bash
curl http://127.0.0.1:3000/tasks/1
```

### Update

```bash
curl -X PUT http://127.0.0.1:3000/tasks/1 \
  -H "Content-Type: application/json" \
  -d '{"completed":true}'
```

### Delete

```bash
curl -X DELETE http://127.0.0.1:3000/tasks/1
```
